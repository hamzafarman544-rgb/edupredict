### Prediction Data Flow
```
Frontend Form вҶ’ POST /api/predictions (Node.js)
    вҶ’ Validate + find student in MongoDB
    вҶ’ POST /predict (Flask ML service)
    вҶ’ Random Forest model returns: prediction, probabilities, feature importance
    вҶ’ Store result in MongoDB Predictions collection
    вҶ’ Return full result to Frontend
```

## Setup & Running (Local Development)

### Prerequisites
- Node.js 20+
- Python 3.11+
- MongoDB 7.0 (or MongoDB Atlas)

---

### 1. ML Service

```bash
cd ml-service

# Install Python dependencies
pip install -r requirements.txt

# Train the model (generates model/model.pkl)
python train_model.py

# Start Flask API
python app.py
# вҶ’ Running on http://localhost:5001
```

**Verify ML service:**
```bash
curl http://localhost:5001/health
# { "status": "ok", "model_loaded": true, "accuracy": 0.914 }
```

---

### 2. Backend

```bash
cd backend

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Edit .env вҶ’ set MONGO_URI, JWT_SECRET, ML_SERVICE_URL

# Start server
npm run dev   # development (nodemon)
# or
npm start     # production

# вҶ’ Running on http://localhost:4000
```

---

### 3. Frontend

```bash
cd frontend

# Install dependencies
npm install

# Configure environment
cp .env.local.example .env.local
# NEXT_PUBLIC_API_URL=http://localhost:4000/api

# Start dev server
npm run dev
# вҶ’ Running on http://localhost:3000
```

---

### Docker (Full Stack)

```bash
# From project root
docker-compose up --build

# Services:
# Frontend  вҶ’ http://localhost:3000
# Backend   вҶ’ http://localhost:4000
# ML Svc    вҶ’ http://localhost:5001
# MongoDB   вҶ’ localhost:27017
```

## Environment Variables

### Backend (.env)
| Variable          | Default                           | Description                     |
|-------------------|-----------------------------------|---------------------------------|
| `PORT`            | `4000`                            | Server port                     |
| `MONGO_URI`       | `mongodb://localhost:27017/edupredict` | MongoDB connection string  |
| `JWT_SECRET`      | вҖ”                                 | Secret for JWT signing (required)|

| `ML_SERVICE_URL`  | `http://localhost:5001`           | Flask ML service URL            |

## API Reference

### Auth Endpoints

| Method | Route                           | Auth | Description               |
|--------|---------------------------------|------|---------------------------|
| POST   | `/api/auth/register`            | No   | Register new user         |
| POST   | `/api/auth/login`               | No   | Login, returns JWT        |
| GET    | `/api/auth/me`                  | JWT  | Get current user          |
| POST   | `/api/auth/forgot-password`     | No   | Send reset link           |
| POST   | `/api/auth/reset-password/:token` | No | Reset password with token |

### Student Endpoints

| Method | Route                  | Auth | Description         |
|--------|------------------------|------|---------------------|
| GET    | `/api/students`        | JWT  | List students       |
| POST   | `/api/students`        | JWT  | Create student      |
| GET    | `/api/students/:id`    | JWT  | Get one student     |
| PUT    | `/api/students/:id`    | JWT  | Update student      |
| DELETE | `/api/students/:id`    | JWT  | Delete student      |

### Prediction Endpoints

| Method | Route                       | Auth | Description                    |
|--------|-----------------------------|------|--------------------------------|
| POST   | `/api/predictions`          | JWT  | Run prediction (calls Flask)   |
| GET    | `/api/predictions`          | JWT  | List predictions (paginated)   |
| GET    | `/api/predictions/stats`    | JWT  | Aggregated stats by class      |
| GET    | `/api/predictions/:id`      | JWT  | Get single prediction          |
| DELETE | `/api/predictions/:id`      | JWT  | Delete prediction              |

### ML Service Endpoints

| Method | Route              | Description                          |
|--------|--------------------|--------------------------------------|
| GET    | `/health`          | Health check + model status          |
| POST   | `/predict`         | Single student prediction            |
| POST   | `/predict/batch`   | Batch prediction for multiple students|
| GET    | `/model/info`      | Model metadata + feature importances |

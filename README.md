## System Architecture

```
┌─────────────────────────────────────────────────────────────────────┐
│                         MICROSERVICES ARCHITECTURE                  │
├──────────────────┬──────────────────┬───────────────────────────────┤
│   FRONTEND       │   BACKEND        │   ML MICROSERVICE             │
│   Next.js 14     │   Node.js/Express│   Python Flask                │
│   Tailwind CSS   │   JWT Auth       │   Random Forest               │
│   Recharts       │   Mongoose       │   scikit-learn                │
│   Port 3000      │   Port 4000      │   Port 5001                   │
└──────────┬───────┴────────┬─────────┴──────────────────────────────┘
           │                │
           │   REST API     │   HTTP POST /predict
           │◄──────────────►│──────────────────────────►
           │                │                           │
                        MongoDB                     model.pkl
                        Port 27017
```

### Prediction Data Flow
```
Frontend Form → POST /api/predictions (Node.js)
    → Validate + find student in MongoDB
    → POST /predict (Flask ML service)
    → Random Forest model returns: prediction, probabilities, feature importance
    → Store result in MongoDB Predictions collection
    → Return full result to Frontend
```

## Setup & Running (Local Development)

### Prerequisites
- Node.js 20+
- Python 3.11+
- MongoDB 7.0 (or MongoDB Atlas)

---

### 1. ML Service

```bash
cd ml-service

# Install Python dependencies
pip install -r requirements.txt

# Train the model (generates model/model.pkl)
python train_model.py

# Start Flask API
python app.py
# → Running on http://localhost:5001
```

**Verify ML service:**
```bash
curl http://localhost:5001/health
# { "status": "ok", "model_loaded": true, "accuracy": 0.914 }
```

---

### 2. Backend

```bash
cd backend

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Edit .env → set MONGO_URI, JWT_SECRET, ML_SERVICE_URL

# Start server
npm run dev   # development (nodemon)
# or
npm start     # production

# → Running on http://localhost:4000
```

---

### 3. Frontend

```bash
cd frontend

# Install dependencies
npm install

# Configure environment
cp .env.local.example .env.local
# NEXT_PUBLIC_API_URL=http://localhost:4000/api

# Start dev server
npm run dev
# → Running on http://localhost:3000
```

---

### Docker (Full Stack)

```bash
# From project root
docker-compose up --build

# Services:
# Frontend  → http://localhost:3000
# Backend   → http://localhost:4000
# ML Svc    → http://localhost:5001
# MongoDB   → localhost:27017
```

## API Reference

### Auth Endpoints

| Method | Route                           | Auth | Description               |
|--------|---------------------------------|------|---------------------------|
| POST   | `/api/auth/register`            | No   | Register new user         |
| POST   | `/api/auth/login`               | No   | Login, returns JWT        |
| GET    | `/api/auth/me`                  | JWT  | Get current user          |
| POST   | `/api/auth/forgot-password`     | No   | Send reset link           |
| POST   | `/api/auth/reset-password/:token` | No | Reset password with token |

### Student Endpoints

| Method | Route                  | Auth | Description         |
|--------|------------------------|------|---------------------|
| GET    | `/api/students`        | JWT  | List students       |
| POST   | `/api/students`        | JWT  | Create student      |
| GET    | `/api/students/:id`    | JWT  | Get one student     |
| PUT    | `/api/students/:id`    | JWT  | Update student      |
| DELETE | `/api/students/:id`    | JWT  | Delete student      |

### Prediction Endpoints

| Method | Route                       | Auth | Description                    |
|--------|-----------------------------|------|--------------------------------|
| POST   | `/api/predictions`          | JWT  | Run prediction (calls Flask)   |
| GET    | `/api/predictions`          | JWT  | List predictions (paginated)   |
| GET    | `/api/predictions/stats`    | JWT  | Aggregated stats by class      |
| GET    | `/api/predictions/:id`      | JWT  | Get single prediction          |
| DELETE | `/api/predictions/:id`      | JWT  | Delete prediction              |

### ML Service Endpoints

| Method | Route              | Description                          |
|--------|--------------------|--------------------------------------|
| GET    | `/health`          | Health check + model status          |
| POST   | `/predict`         | Single student prediction            |
| POST   | `/predict/batch`   | Batch prediction for multiple students|
| GET    | `/model/info`      | Model metadata + feature importances |

```bash

  -H "Content-Type: application/json" \
```

## ML Model Details

| Property         | Value                         |
|------------------|-------------------------------|
| Final model      | Random Forest Classifier      |
| n_estimators     | 200                           |
| max_depth        | 10                            |
| Training samples | 800 (80/20 split)             |
| Test accuracy    | ~91.4%                        |
| Weighted F1      | ~0.90                         |
| Classes          | High / Average / At Risk      |

### Input Features
| Feature           | Range    | Description                      |
|-------------------|----------|----------------------------------|
| attendance        | 0–100    | Attendance percentage            |
| grade_avg         | 0–100    | Average grade across subjects    |
| assignments       | 0–100    | Assignment completion rate       |
| engagement        | 0–100    | Engagement score                 |
| prev_gpa          | 0.0–4.0  | Previous GPA                     |
| study_hours       | 0–12     | Daily study hours                |
| participation     | 0–10     | Class participation score        |
| missed_deadlines  | 0–7      | Number of missed deadlines       |

### Models Compared
| Model               | Accuracy | F1   | Notes               |
|---------------------|----------|------|---------------------|
| Logistic Regression | ~83%     | 0.82 | Baseline            |
| Decision Tree       | ~87%     | 0.86 | Interpretable       |
| **Random Forest**   | **~91%** |**0.90**| **Final model** |
